# hugo-tweet

> **Warning** Work in progress

Hugo theme component to display static tweets.

## Features

- [x] Text + Emojis
- [x] Images
- [x] videos
- [x] polls
- [x] Quote tweets
- [x] Opengraph Cards
- [ ] Mixed Media
- [ ] Twemoji
- [ ] Private Use Area Glyphs
- [ ] Hashflags
- [ ] Profile pics with shapes
- [ ] Affiliate Badges
- [ ] Backup data to json file
- [ ] CSS Preflight 
- [ ] Most Ads eg: 3d ads
- [ ] Unified Cards

## Running the example locally

```sh
pnpm install
pnpm dev
```
